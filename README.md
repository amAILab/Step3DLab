# Step3D Lab preview

Новый hosting-ready preview для GitHub Pages: <https://amailab.github.io/Step3DLab/>.

## Состав

- `index.html` — новая главная страница с лендингом Step3D Lab.
- `styles.css` — адаптивная визуальная система без сборки.
- `script.js` — мобильное меню и быстрый бриф с копированием текста заявки.
- `photos_do_not_touch/` и `assets/` — используемые изображения и базовые ассеты.
- `robots.txt` + meta `noindex,nofollow` — preview закрыт от индексации.

Цены намеренно не указаны: расчёт зависит от геометрии, материала, количества, сроков, точности и постобработки.
